# DECISIONS

## 为什么不把Excel直接放向量库

原因：

录取位次、分数线、招生计划属于：

结构化数值数据。

需要支持：

* 排序
* 过滤
* 统计
* 位次换算

因此：

Excel -> 数据库

而不是：

Excel -> 向量库。

---

## 为什么先使用SQLite

当前项目仍处于：

单机开发阶段。

SQLite：

* 简单
* 无需部署
* 易迁移
* 适合原型开发

后期可迁移 PostgreSQL。

---

## 为什么使用Docker

原因：

保证：

* 多电脑环境一致
* 依赖一致
* Python版本一致
* Codex运行环境一致
